# 📖 Glossário

Aqui você encontra os principais termos utilizados na Mahalo.

| Termo           | Significado                                                         |
| --------------- | ------------------------------------------------------------------- |
| **RP**          | Roleplay - interpretar um personagem dentro do servidor             |
| **IC**          | In Character - tudo que acontece dentro do personagem               |
| **OOC**         | Out of Character - fora do personagem, comunicação real             |
| **NLR**         | New Life Rule - após morrer, seu personagem esquece o que aconteceu |
| **RDM**         | Random Deathmatch - matar sem motivação de RP                       |
| **VDM**         | Vehicle Deathmatch - usar veículo como arma sem motivação de RP     |
| **Metagame**    | Usar informações OOC para benefício IC                              |
| **Powergaming** | Forçar ações impossíveis ou não dar chance ao outro jogador         |
| **Failrp**      | Agir de forma que quebra a imersão do roleplay                      |
| **Combatlog**   | Desconectar durante situações de conflito para evitar consequências |
| **Staff**       | Equipe administrativa do servidor                                   |
| **Whitelist**   | Cargo que requer aprovação para ser acessado                        |
